### Setup
* rename the `.env-example` file to `.env` and fill the appropriate value to connect to MongoDB.
```
cd Scaffoldzoi
yarn
```

### Run Server
```
cd packages/server
yarn start
```

### Run Client
```
cd packages/client
yarn start
```
